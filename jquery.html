jQuery

Write less, do more

O que é?

jQuery é uma biblioteca JavaScript multi browser, desenvolvida para simplificar a forma como interagimos com o Document Object Model (DOM), como tratamos eventos e como realizamos a comunicação entre o client e o servidor.

jQuery nos ajuda a:

  • Encontrar elementos em um documento HTML
  • Alterar o conteúdo de um documento HTML
  • Identificar o que o usuário faz e agir de acordo
  • Animar o conteúdo em uma página
  • Comunicar através da rede para buscar mais conteúdo
